## useDebounce

This hook utilizes the `requestAnimationFrame` to debounce performance-heavy functions.

You can pass following options via parameter object

| Key       | Type     | Description                                  |
| :-------- | :------- | :------------------------------------------- |
| callback  | function | Performance-heavy function we need debounce  |

The return object includes following properties

| Key       | Type      | Description                                                |
| :-------- | :-------- | :--------------------------------------------------------- |
| callback  | function  | Passed callback function wrapped in the debounce mechanism |

```js
import { useDebounce } from '@securityscorecard/design-system';

const debouncedFn = useDebounce(heavyFn);
```
